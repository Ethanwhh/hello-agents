# 赛博小镇 - AI NPC对话系统

基于HelloAgents框架的AI小镇模拟游戏,展示多智能体系统在游戏中的应用。

## 🎮 功能特性

- ✅ 3个AI NPC (张三、李四、王五)
- ✅ 智能对话系统
- ✅ 记忆系统 (短期+长期记忆)
- ✅ 好感度系统 (5个等级)
- ✅ NPC自主行为 (闲逛、工作)
- ✅ 完整的日志系统

## 🛠️ 技术栈

- **游戏引擎:** Godot 4.x
- **后端框架:** FastAPI + Python 3.10+
- **AI框架:** HelloAgents
- **LLM:** OpenAI GPT-4 (可配置其余的LLM服务)
